Réseaux sans fil disponibles
Si votre environnement comprend plusieurs réseaux sans fil, cet écran peut lister plusieurs réseaux sans fil. Sélectionnez le nom du réseau sans fil approprié puis cliquez sur “Suivant”. Si vous ne savez pas quel réseau sans fil choisir, veuillez demander conseil à votre administrateur système.

Si votre réseau sans fil n'est pas listé, cliquez sur le bouton “Actualiser” pour relancer une recherche des réseaux sans fil dans votre environnement. Si vous ne voyez toujours pas de réseaux sans fil et que vous êtes sûr que le nom du réseau (SSID) n'est pas caché, veuillez effectuer les vérifications suivantes :
  1. Vérifiez que votre imprimante/appareil multifonction est allumé.

  2. Essayez de rapprocher votre appareil du point d'accès ou de votre ordinateur.

  3. (Uniquement applicable aux utilisateurs qui n’emploient pas de câble USB pour configurer leur périphérique d’impression sans fil). Vous pouvez temporairement désactiver tout programme de pare-feu s’exécutant sur votre ordinateur. En effet, il peut empêcher votre ordinateur de détecter les réseaux sans fil. Si vous choisissez de désactiver temporairement votre pare-feu, il est recommandé de déconnecter votre point d’accès de l’Internet pour éviter les failles de sécurité dans votre ordinateur.
Si vous avez configuré votre point d'accès pour cacher son nom SSID, vous devez entrer le nom du réseau manuellement en cliquant sur le bouton “Avancé”.

Veuillez noter : l'intensité du signal sans fil correspond à l'intensité du signal entre votre imprimante/appareil multifonction et le point d'accès avec lequel il communique.
